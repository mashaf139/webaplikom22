Sistem Operasi

Dapat dikatakan bahwa sistem operasi bereaksi terhadap proses tertentu dengan mengendalikan semua sumber daya sistem dan menawarkan berbagai layanan kepada pengguna untuk memfasilitasi dan mengamankan penggunaan. Sistem operasi memiliki kemampuan untuk mengelola dan melacak bagaimana berbagai program aplikasi dan pengguna menggunakan perangkat keras. Sistem operasi Windows adalah sistem operasi berbasis Graphical user interface (GUI), sehingga mudah digunakan. Sistem operasi mempunyai peran penting di dalam suatu sistem komputer. Sistem operasi berfungsi sebagai kerangka kerja untuk semua program yang diinstal pada perangkat. Sistem operasi bertanggung jawab untuk mengoordinasikan semua perangkat yang terhubung secara bersamaan, termasuk CPU, speaker, mouse, dan penyimpanan internal. Sistem operasi memiliki kemampuan untuk memaksimalkan kinerja baik perangkat keras maupun perangkat lunak. Tentu saja, sistem operasi memungkinkan semua perangkat untuk bekerja sama satu sama lain dan bekerja sebagai satu unit secara maksimal. Sistem operasi merupakan salah satu komponen perangkat lunak komputer dalam sebuah sistem komputer. Sebagai pengganti sistem komputer mekanik, generasi pertama menandai dimulainya perkembangan sistem komputasi elektronik. Generasi ini sistem komputer belum dilengkapi sistem operasi. Tetapi beberapa fungsi sistem operasi telah ada, contohnya fungsi sistem operasi ialah FMS dan IBSYS. Sistem operasi pada generasi ini dikembangkan untuk melayani banyak pengguna. Pada masa ini sistem operasi telah menggunakan GUI (Graphical User Interface), yaitu antar-muka komputer yang berbasis grafis yang sangat nyaman dan mudah digunakan. Pada generasi ini mulai diperkenalkan sistem operasi yang bisa digunakan pada perangkat lain.

Microsoft word

Multi-Tool Word untuk Xenix pertama kali dirilis pada tahun 1983, dan versi yang lebih baru dibuat untuk sejumlah sistem operasi yang berbeda, termasuk DOS (1983), Apple Macintose (1984), SCO UNIX, OS/2, dan Microsoft Windows (1989) . Itu diberi moniker Microsoft Office Word setelah diintegrasikan ke dalam Sistem Microsoft Office 2013 dan 2007. Program pengolah kata ini dapat membantu menyelesaikan tugas apa pun yang melibatkan dokumen, pesan, dan sejenisnya.

Microsoft Excel

Memahami alat atau aplikasi yang dikenal dengan Microsoft Excel disertakan dalam paket instalasi untuk Microsoft Office, dan menggunakan spreadsheet dengan baris dan kolom, proses angka. Apple Spreadsheet adalah alat manajemen data utama yang digunakan oleh Microsoft Excel, sedangkan rumus Excel digunakan untuk melakukan aktivitas Excel yang lebih umum. Pengertian Microsoft Office dan Microsoft Excel adalah kumpulan perangkat lunak produktivitas kantor yang dapat digunakan dengan sistem operasi Windows dan Mac OS X. Dibandingkan dengan aplikasi lain, Microsoft Office memiliki antarmuka yang sangat ramah pengguna dan menawarkan berbagai jenis pengeditan file.

Google Slide

Aplikasi Pengolah Bahan Tayang Aplikasi pengolah bahan tayang adalah suatu aplikasi yang digunakan untuk membuat file presentasi dengan tampilan slide show. Program ini dirancang untuk memudahkan Anda membuat presentasi berkualitas profesional dengan menggunakan berbagai template dan animasi. Anda dapat membuat presentasi dengan cepat menggunakan ratusan template Visme dan fitur pengeditan grafik online. Anda dapat memanfaatkannya dengan membuka langsung situs web canva.com dan mulai membuat slide presentasi yang Anda inginkan. Anda tidak perlu mencari foto dan gif di tempat lain karena slidebean ini memiliki banyak fitur dan terintegrasi dengan Giphy dan Unsplash. Ini memiliki banyak fitur yang memungkinkan pembuatan dokumen presentasi dengan cepat dan efektif. Menggunakan program desktop yang dapat Anda unduh dan instal di perangkat Anda, Anda dapat menggunakan aplikasi ini secara offline. Popularitas menyajikan aplikasi seperti Google Slides tidak diragukan lagi. Banyak fungsi dalam program ini disertakan dalam Powerpoint. Keuntungannya, selama Anda memiliki akun Google, Anda dapat mengaksesnya secara gratis di smartphone. Slide Google ini dapat dimodifikasi bersama secara online, menjadikannya ideal untuk presentasi kolaboratif dan pengguna juga dapat mengunduh dokumen atau file presentasi dalam beragam versi seperti Microsoft PowerPoint, dokumen PDF, gambar PNG, dan lainnya.

Figam

Figma adalah tool desain berupa website yang terhubung dengan cloud sehingga bisa digunakan kapanpun dan dimanapun melalui internet. Tool ini berbasis vector, sehingga akan lebih cocok untuk mendesain UI website atau mobile dan aset ilustrasi.
